## üîπ Overview
A full-featured **LinkedIn Clone** built with the **MERN stack** that replicates core linkedin professional networking features. The platform enables users to create profiles, share content, connect with professionals, and engage in real-time communication.

## üöÄ Implemented Features

### Authentication & User Management
- User registration with email verification
- Secure login with JWT authentication
- Protected routes and API endpoints
- Session management

### User Profiles
- Complete profile creation and editing
- Professional headline and summary
- Work experience tracking
- Education history
- Skills and endorsements
- Profile picture and cover photo uploads
- Social media links integration

### Posts & Content
- Create and share rich text posts
- Image and media uploads in posts
- Like and comment on posts
- Post sharing and saving
- Hashtag support
- Delete and edit posts

### Networking & Connections
- User search and discovery
- Send/accept/reject connection requests
- View mutual connections
- Connection suggestions
- Profile views tracking

### Real-time Messaging
- One-on-one chat
- Message history
- Online/offline status
- Read receipts
- Typing indicators
- File and media sharing

### Notifications
- Real-time in-app notifications
- Email notifications for:
  - Welcome email notification
  - New connection accept/request email notification
  - Post likes and comments email notification
  - New Messages notification
  - Profile views notification
- Notification settings
- Mark as read/unread

### Search & Discovery
- Search users by name, skills, or company
- Advanced search filters
- Recent searches
- Search suggestions
